# calendar
目前已实现的功能：
  1.显示日历，日历的日期选择，月份切换的基础功能
  2.添加待办事项，待办事项的日期绑定（点击日历的日期进行时间绑定），代办事项根据日期显示（只显示当天对应的事项）
  
ps：本项目是自己在学习Vue框架的过程中的动手练习，借鉴了一些vue日历插件的实现思路以及todo-list项目的思路，目前还在添加和完善更多功能，一边学一边做，有时间就会更新
